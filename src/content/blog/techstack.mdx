# Development Blog
### 2026-03-26

This is a technical blog for those interested in development and programming. 

As Arraz was just a quick side project, I decided to go with a tech stack I already knew well:
- React
- TypeScript
- TanStack Router
- Vite
- Tailwind

These are all tools that make development incredibly fast and time-saving, and they're also somewhat universally known
among the interested. This makes the stack optimal for an open source, simple side project. 

Tailwind is by far the biggest timesaver here, and is what I've been using in every web-project I've made for years. I really 
can't imagine writing styling any other way now. This allows me to quickly whip up new games and focus more on the game logic
than jumping between files.

TanStack router has also become a favorite. File-based routing is definitely my favorite way, and TanStack has some seriously 
neat features. Since Arraz is relatively simple and minimalist, I don't necessarily need most of them, but they're really 
nice to keep in mind. 

As for future expansion: if I ever decide to create an account system (optional, of course), I'd love to try making the backend in Go.
For a while now I've been writing everything in TS, so learning a new backend language might be about time.
