## About The Project
I created Arraz as I found most existing scorekeeping webapps for darts either outdated, broken, or bloated. Arraz was created to be simple, straight forward and minimal.

## Getting Started

To run Arraz locally, you'll need [Bun](https://github.com/oven-sh/bun?tab=readme-ov-file#install) installed.

1. `git clone https://github.com/mariuseikenes/arraz`

2. `bun install`

3. `bun dev` (or `bun dev --host` if you want to access Arraz on another device locally)
